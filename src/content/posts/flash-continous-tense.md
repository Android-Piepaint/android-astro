# Part 1

你在漆黑的熒幕上</br>
注視著什麼呢？</br>
</br>
遠處的山崗上</br>
傳來了松果香[^1]</br>
</br>
</br>
今天也和往常一樣，在痛苦、憂愁中過去</br>
時間也不知不覺</br>
從中流逝而過</br>
</br>
</br>
在我的記憶中，這一成不變的世界</br>
是由專有軟體組成的「圍牆花園」</br>
</br>
</br>
我們一起奔向那自由的彼方</br>
和專有軟體道過「來日見」後永不再見面</br>
因爲未發現自由替代，所以與家人失聯</br>
就這麼記住，第一次逃離「圍牆花園」[^2]的實踐</br>

![Oneplus Ace 5 running Android 16 GSI](assets/oneplus-ace5-a16-gsi.jpg)

</br>
</br>

早就 習慣 專有軟體</br>
（甚至）不理解，什麼是「自由」
</br>
</br>
</br>

# Part 2

我在不知不覺中</br>
獨自在天臺徘徊</br>
並幻想着把手機改造自由軟體的未來</br>
終於有一天，幻想成真</br>
手機「解鎖」是前提</br>
鍵入命令，電源鍵確認</br>
「圍牆花園」，再見！</br>
</br>
![Oneplus 6T and Mi pad5 running mainline Linux](assets/nabu.jpg)
</br>
開機之後，日誌滾動</br>
早已把我震驚</br>
想不到初次刷機的過程竟如此順利</br>
</br>
</br>
只是因「專有軟體」摧殘，手機早已「命不久矣」</br>
當年的刷機少年如今長大成人</br>
即使黑磚、Crashdump 也沒關係，刷機還是進行式</br>
如果傷心的話，就再換手機吧</br>
一切又從頭開始</br>
</br>
</br>
圍牆的 那邊是 自由的</br>
我終於 找到 棲身之處</br>
</br>

# Part 3

手機到家，一切又重新開始</br>
「解鎖」條件，似乎又提高不少</br>
長期經歷，（問題）早已熟練應對</br>
在 Linux 中開懷大笑</br>

# Ending

我們一起奔向那自由的彼方</br>
和專有軟體道過「來日見」後永不再見面</br>
即使不知機理也無所謂啊，只需記住那天</br>
遙遠的記憶之中，當初的勇敢實踐</br>
</br>
</br>
依靠著 專有 軟體</br>
只在我 理解 自由 之前</br>
現在 隱私 已被我 掌控</br>
來吧，再次開懷大笑</br>

[^1]:指香港生產搭載 Linux 的手機、平板電腦、筆電等的「PINE64」公司。因爲其 LOGO 爲松果，故稱之為「松果香」。
[^2]:指由 Android 和 iOS 以及其它專有軟體組成的軟體使用環境。這是現在智慧型手機的普遍軟體特徵。
